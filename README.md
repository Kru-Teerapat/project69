# ระบบรายงานแผนงานโครงการ โรงเรียนครุศาสตร์

Dashboard สำหรับผู้บริหารเพื่อติดตามความคืบหน้าของโครงการต่างๆ ข้อมูลอัปเดตแบบเรียลไทม์จาก Google Sheets

## ฟีเจอร์
- แสดงจำนวนโครงการ, งบประมาณรวม, ใช้ไปแล้ว, และคงเหลือ
- แสดงกราฟสถานะโครงการและสัดส่วนงบประมาณตามกลุ่มงาน
- ตารางสรุปข้อมูลพร้อมฟังก์ชันค้นหา
- ดึงข้อมูลอัตโนมัติจาก Google Sheet ทันทีที่โหลดหน้าเว็บ

## วิธีการนำขึ้น GitHub เพื่อเผยแพร่ (เนื่องจากระบบของคุณยังไม่ได้ติดตั้ง Git)

1. ไปที่เว็บไซต์ [GitHub](https://github.com/) และล็อกอินเข้าสู่ระบบ
2. คลิกปุ่ม **New** (หรือ + มุมขวาบน -> New repository) เพื่อสร้าง Repository ใหม่
3. ตั้งชื่อ Repository เช่น `edu-dashboard`
4. ทำให้ Repository เป็น **Public** แล้วคลิก **Create repository**
5. ในหน้า Repository ที่สร้างเสร็จแล้ว ให้คลิกลิงก์ **"uploading an existing file"** (อยู่บริเวณกลางหน้าจอ)
6. ลากไฟล์ในโฟลเดอร์นี้ทั้งหมดไปวาง ได้แก่:
   - `index.html`
   - `style.css`
   - `script.js`
   - `README.md`
7. พิมพ์ข้อความในช่อง Commit changes แล้วคลิกปุ่ม **Commit changes**
8. ไปที่แท็บ **Settings** ของ Repository 
9. ที่เมนูด้านซ้าย เลือก **Pages** (อยู่ในหมวด Code and automation)
10. ในส่วน Build and deployment ใต้หัวข้อ Source ให้เลือก **Deploy from a branch**
11. ใต้หัวข้อ Branch ให้เลือก **main** หรือ **master** (ขึ้นอยู่กับ branch เริ่มต้นของคุณ) แล้วกด **Save**
12. รอประมาณ 1-2 นาที จากนั้นรีเฟรชหน้าต่าง คุณจะเห็นลิงก์เว็บไซต์ Dashboard ของคุณปรากฏขึ้นมา (เช่น `https://[ชื่อผู้ใช้].github.io/edu-dashboard/`)
